# 个人作品集网站 PRD

**版本**: v1.0 / **日期**: 2026-05-05 / **作者**: 蒋海崇

---

## 1. 项目目标

打造一个让人**看着舒服**的个人作品集网站，清晰展示个人信息、经历、技能和项目，给访客留下专业且亲切的印象。

---

## 2. 核心功能

| 功能 | 说明 |
|------|------|
| 个人信息展示 | 姓名、头衔、简介、头像 |
| 个人经历 | 工作经历 + 教育背景，时间线展示 |
| 技能展示 | 技术栈分类展示 |
| 项目展示 | 项目卡片（名称、描述、技术栈、链接） |
| 联系方式 | 邮箱、GitHub 等 |
| 响应式设计 | 手机 + 电脑都能舒适浏览 |

---

## 3. 设计风格

**核心理念：舒服、干净、温暖**

- **风格**: 简约温和，不炫技，注重内容的呼吸感和阅读舒适度
- **配色**: 暖白/米白底色 + 柔和的点缀色，低对比度，护眼友好
- **字体**: 圆润的无衬线字体
- **布局**: 单页滚动，留白充裕，卡片圆角柔和
- **动效**: 轻微的渐入过渡，不打扰浏览节奏
- **主题**: 默认浅色，可选深色模式

---

## 4. 页面结构

导航 → Hero（头像 + 一句话介绍） → 个人经历（时间线） → 技能 → 项目 → 联系方式 → Footer

---

## 5. 技术方案

- **框架**: Next.js + TypeScript
- **样式**: Tailwind CSS
- **动画**: Framer Motion（轻量使用）
- **数据**: 集中配置在数据文件中，方便后续更新
- **部署**: Vercel / GitHub Pages
